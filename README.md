## 📖 Project Overview

**DREAM 11** is a dynamic and interactive React-based web application that allows users to build their ultimate fantasy cricket team. Users start by claiming virtual currency, which they can then use to strategically draft a squad of top-tier players. The application features seamless state management, strict validation rules for team building, and a professional, user-friendly interface.

## ✨ Key Features

- **Interactive Virtual Economy:** Users can claim free coins via the interactive banner to build their budget. The virtual balance dynamically updates whenever a player is drafted or removed.
- **Robust Squad Management & Validation:** Seamlessly toggle between "Available Players" and "Selected Squad" views. The system strictly enforces a maximum 6-player limit and prevents duplicate player selections.
- **Real-Time Notifications:** Replaced standard browser alerts with sleek, customized `React-Toastify` pop-ups for actions like insufficient funds, successful drafts, and validation warnings.
- **Responsive UI/UX:** A fully responsive design featuring a fixed Navbar displaying the live coin balance, detailed player cards (price, role, style, rating), and an engaging Newsletter subscription section.

## 🛠️ Technologies Used

- **Frontend Core:** React.js, HTML5, CSS3
- **Styling:** Tailwind CSS (via DaisyUI/Custom classes)
- **Build Tool:** Vite
- **Packages:** `react-toastify` (for toast notifications)
- **Data Management:** Local JSON (simulated API for 15 players), LocalStorage (for optional newsletter subscription state)
